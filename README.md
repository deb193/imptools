# imptools
Imputation Tools

## Installation

Run the following:

``` r
install.packages("devtools")
devtools::install_github("dsquire/imptools")
```
If you get an error when using the command above you can also try:

``` r
install.packages("devtools")
devtools::install_github("dsquire/imptools", ref = "main")
```

``` r
library("imptools")
```
